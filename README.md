# RPG
RPG é um jogo de rpg de texto desenvolvido em java com swing e arquivos json para a configuração dos elementos do jogo.

## Como rodar
**PS: É necessário ter o Apache Maven instalado no computador para poder buildar o programa**
* Faça o download do projeto
* Abra o prompt de comando e dê cd até a pasta **\Rpg** e faça o seguinte comando `mvn clean install`
* Depois que o Maven terminar o processo dê cd até a pasta **\Rpg\target** e rode o comando `java -jar Rpg-1.0.0-jar-with-dependencies.jar`
* Está tudo pronto
